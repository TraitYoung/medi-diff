## 岗位职责

1. 负责 AI 急救产品的端到端开发（前端 + 后端 + 系统架构），支撑院前急救、调度指挥、现场辅助决策等核心业务场景。

2. 构建支持文本/语音/图像/视频的多模态 AI 系统，实现对急救现场信息（如伤情描述、生命体征、环境图像）的实时感知与理解。

3. 设计与实现多 Agent 协作机制（任务拆解、调度、通信），用于急救资源调度、指令分发、急救指导等协同任务。

4. 搭建高效的数据管道与 RAG 系统，整合急救知识库、历史案例、医疗指南，为 Agent 提供实时、精准的知识检索。

5. 优化模型推理效果（Prompt / Tool / Memory / Workflow），保障在低延迟、高并发、弱网环境下的系统可用性。

6. 持续提升系统性能、稳定性与用户体验，满足急救场景对高可靠、高实时、可解释性的要求。
