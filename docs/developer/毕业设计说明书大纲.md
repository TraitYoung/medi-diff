## 第 1 章　绪论

### 1.1 研究背景与意义

乳腺钼靶X线摄影是乳腺癌筛查的首选影像学方法，其图像质量的稳定性和数据多样性对提升AI辅助诊断模型的泛化能力与鲁棒性至关重要。然而，高质量、标注详尽且涵盖广泛解剖变异与病理类型的临床影像数据难以大规模获取，面临隐私、成本及类别不均衡等限制。传统的生成对抗网络（GAN）在生成高分辨率、细节丰富的医学图像时，常面临模式崩溃与训练不稳定的挑战。扩散模型作为新一代生成式AI的代表，通过渐进式去噪过程能生成质量更高、多样性更丰富的图像。本设计旨在构建一个基于扩散模型的乳腺钼靶图像生成系统，能够合成高度逼真且解剖结构合理的多样化乳腺影像，特别是针对特定解剖特征（如致密型乳腺）或病理模式（如早期微钙化），为下游诊断算法训练提供安全、可控的扩展数据源，具有显著的科研价值与临床应用潜力。

### 1.2 国内外研究现状

- 医学图像生成技术演进：GAN → VAE → 扩散模型（DDPM → LDM → Stable Diffusion）
- 扩散模型在医学影像中的代表性工作：Med-LVDM、目标引导跨模态翻译、3D 医学图像生成、混合量子-经典 LDM 等
- 条件生成技术：文本描述条件注入、语义图引导、解剖约束
- 乳腺影像生成的特殊挑战：高分辨率需求、解剖结构保真度、文本伪影抑制
- 主要参考文献见任务书及正文参考文献章节

### 1.3 主要研究内容

依据任务书「设计工作内容」，本设计围绕以下六个方面展开：

1. **前沿技术调研与方案论证**：深入研究扩散模型（DDPM、Stable Diffusion、LDM）及条件生成技术在医学影像领域的最新应用进展。对比分析扩散模型与GAN在医学图像生成任务上的性能、稳定性与计算成本，论证技术选型的合理性。完成详尽的文献综述与技术路线设计报告。

2. **数据集构建与预处理流水线开发**：获取并管理公开乳腺钼靶数据集（CBIS-DDSM），进行脱敏与格式统一。设计并实现完整的预处理流水线，包括图像标准化、ROI提取、异常值处理，以及结合临床报告的标签提取与结构化。构建条件-图像对用于模型训练。

3. **条件扩散模型的实现与训练**：使用PyTorch框架搭建条件潜在扩散模型，基于 Stable Diffusion 1.5 + LoRA 设计并编码 UNet 去噪网络与条件注入机制。实现并调试训练循环，集成混合损失函数（噪声预测损失、感知损失），应用梯度裁剪、EMA 等稳定训练的技术。在划分的训练集与验证集上进行多轮训练与迭代调优。

4. **端到端系统集成与部署**：利用 FastAPI 开发高性能后端 RESTful API，封装模型推理、图像后处理及评估计算功能。采用 Gradio 快速构建直观的前端交互界面，集成条件输入、任务队列、结果画廊与评估报告展示等功能模块。将数据预处理、模型服务、后处理流水线及用户界面无缝集成，确保系统可一键部署与运行。

5. **系统化评估与消融实验**：设计全面的实验方案，在保留的测试集上定量评估生成图像的多样性与保真度。构建 16 维域内规则评审体系（A–F 组）与 FID/KID/PRC/sFID 通用学术指标双轨并行。进行消融实验，分析不同 LoRA 版本、强度参数、引导系数、调度器等对生成效果的具体影响。

6. **设计文档编纂与成果总结**：撰写系统架构设计文档、API 接口文档及用户操作手册。严格按照规范完成毕业论文，重点论述模型实现细节、实验分析与结论。

## 第 7 章　多维图像质量评估体系

### 7.1 评估体系设计思路
- 双轨指标互补：域内规则 + 通用学术指标
- 为什么不能只用 FID：Inception 特征对灰阶钼靶的域偏移

### 7.2 域内规则指标体系
- **A 组（构图与解剖）**：圆形度、轮廓凹陷度、主体面积比
- **B 组（灰阶统计）**：均值、标准差、直方图分布匹配
- **C 组（纹理与频率）**：边缘密度、功率谱 β、Gabor 响应
- **D 组（伪影检测）**：条带伪影、气泡、过曝、接缝
- **E 组（真实分布偏差）**：与真实钼靶基线各分位偏差
- **F 组（无参考感知）**：BRISQUE、NIQE
- 硬标签一票否决机制（BANDING、SHAPE_ODD、ARTIFACT_BUBBLES 等）

### 7.3 通用深度学习指标
- FID / KID：整体分布距离
- Precision / Recall / F-score（PRC）：流形覆盖度
- sfid_spatial768：空间 patch 分布的 Fréchet 距离

### 7.4 评分聚合与校准
- 维度权重设计（A=0.20, B=0.10, C=0.20, D=0.25, E=0.05, F=0.20）
- 真实基线自动校准（`--real-images-dir`）
- 严格模式 vs 校准模式的双视角输出

## 第 10 章　总结与展望

### 10.1 工作总结
- 主要贡献：全图单次 img2img 管线、16 维规则评审体系、双轨评估框架、答辩级交互界面
- 技术亮点：DPM-Solver++ 高效推理、标签守护后处理、源图质量预筛、多维伪影检测

### 10.2 不足与局限
- 解剖结构保真度仍有提升空间（皮肤线、胸大肌边缘）
- BRISQUE 与纹理增强的 trade-off 未完全解决
- 评估指标基于统计 proxy 而非临床判读
- 未涵盖病理异常生成（肿块、微钙化）

### 10.3 未来展望
- 引入更强的解剖条件约束（ControlNet / T2I-Adapter）
- 频段选择性后处理突破 BRISQUE–纹理 trade-off
- 病理可控生成与反事实推理
- 多模态条件融合（BI-RADS 报告 + 对侧乳腺 + 先验图像）
- 临床专家主观评估验证
