## 评分维度说明

参考 **ACR MQSA（美国放射学院乳腺摄影质量标准）**与 **Sickles 2013** 临床评审规范：

| 维度 | 英文标识 | 评分范围 | 说明 |
|------|----------|----------|------|
| 解剖轮廓合理性 | `anatomy` | 0–5 | 乳腺边缘轮廓连续，MLO 体位可见清晰胸肌线，形态符合正常腺体 |
| 腺体纹理真实性 | `texture` | 0–5 | 纤维腺体密度梯度存在，局部小叶结构可辨，无过平滑/过噪 |
| 亮度与对比度 | `contrast` | 0–5 | 背景区域纯黑（≤5 灰度），腺体区均值合理（30–100），动态范围充足 |
| 伪影/异常缺失 | `artifact` | 0–5 | 无格线、无条纹、无环形异常、无明显拼接痕迹 |
| 总体钼靶相似度 | `overall` | 0–5 | 整体上与真实 FFDM 影像的直观相似程度（最终综合印象） |

**分值参考**：0=完全不合格，1=严重缺陷，2=明显缺陷，3=可接受，4=良好，5=优秀（接近真实影像）

**加权方式**：`总分 = 0.25×anatomy + 0.25×texture + 0.20×contrast + 0.20×artifact + 0.10×overall`（满分 5.0）

## 评分使用方法

### 方式一：脚本交互逐图评分

```bash
python3 scripts/evaluation/simulated_expert_score.py \
  --images-dir outputs/eval/trial_eff_20260503/top_k_images \
  --evaluator "评审人A" \
  --output outputs/eval/trial_eff_20260503/expert_report.md
```

### 方式二：填写下方表格后批量导入

将下表复制为 CSV（保留列头 `filename,anatomy,texture,contrast,artifact,overall`）保存为 `docs/expert_scores_preset.csv`，然后：

```bash
python3 scripts/evaluation/simulated_expert_score.py \
  --images-dir outputs/eval/trial_eff_20260503/top_k_images \
  --scores-csv docs/expert_scores_preset.csv \
  --output outputs/eval/trial_eff_20260503/expert_report.md
```
