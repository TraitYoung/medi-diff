# 开发日志

## 2026-05-18

### 预处理脚本精简

- `clean_labels_inpaint.py`、`clean_text_craft_lama.py`、`label_guard_pick.py`、`enhance_structure.py` 归档至 `archive/preprocessing/`。
- 原因：当前主线只用 `clean_training_labels.py` 产出 CBIS_CLEAN_V2 训练数据，mask 内文字擦除（CRAFT+LaMa / SD Inpaint）属于实验性增强，不在答辩主线内。
- 保留的预处理管线：`clean_cbis.py` → `build_breast_masks.py` → `clean_training_labels.py` → `generate_captions.py`（+ `mammo_label_heuristic.py` 被 `label_guard.py` / `pipeline_orchestrator.py` 引用）。

### 生成质量与源图筛选增强

- 新增 `_has_source_artifact_burden()`：综合 marker 分数、背景标记数、前景亮点数、纹理强度、类病灶模式等多维信号，评估源图伪影负担，低质量源图降级为备选池。
- 新增 `_main_tissue_mask()`：Otsu + 形态学闭运算 + 最大连通域，提取乳腺主体区域掩膜。
- 新增 `_suspicious_lesion_stats()`：检测生成图中的类占位病变（局限肿块、微钙化簇），通过圆形度/实心度/对比度等多重约束降低误报；异常图像自动跳过不保存。
- `filter_source_pool()` 增强：源图分析前统一缩放到 1024px；新增 Laplacian + Sobel 纹理强度指标；artifact_entries 独立分类（伪影源图与优质源图分流，不足时从备选池补充）。
- 生成后 lesion-stats 自检：肿块或钙化簇 > 0 或钙化点 ≥ 4 则跳过当前候选重抽。

### Gradio 界面微调

## 2026-05-14

### Gradio 答辩版界面精简

- 「生成」面板固定使用 `full-image` 全图模式，移除 Patch 拼接入口，降低答辩演示时的解释成本。
- 隐藏 LoRA 路径、LoRA scale 与 Negative Prompt 控件；Gradio 内部固定使用 `outputs/lora/mammo_sd15_v6_allMLO/final_lora`。
- 一键流水线与生成面板保持同一套简化口径：只暴露速度预设、体位、密度、数量、种子、输出前缀与后处理开关。
- 画廊按批次内图片更新时间选择最新批次，避免刷新后仍停留在旧批次。
- 文档同步更新：README、`apps/README.md` 与用户操作手册均标注答辩版前端隐藏高级研究参数；CLI/API 仍保留这些参数用于实验复现。

### 第一轮图像质量调优：保解剖结构

- 将 Gradio 快速/精细预设下调为更保守的 img2img 强度：快速 `strength=0.34, guidance=6.2`，精细 `strength=0.36, guidance=6.5`，优先保留源图中的胸大肌、乳房下皱襞和边界轮廓。
- 内部 prompt 增加精简版 MLO 解剖约束：胸大肌三角形边缘、IMF、从胸壁向乳头方向汇聚的纤维腺体条索和韧带纹理；控制在 CLIP token 限制内，避免关键约束被截断。
- 内部 negative prompt 增加对水墨晕染、云雾状纹理、圆形白点、随机钙化样亮斑、放射/锥形模糊、锯齿撕裂边缘的抑制。
- 后处理后增加二次亮点/金属标记清理，并降低锐化强度，减少纯圆白点和边缘伪影被后处理放大的概率。
- 源图筛选从“按文件大小优先”改为“按真实图像几何质量优先”：实际长宽比异常、主体面积异常、轮廓破碎或凹陷过强的源图会被剔除，避免 RMLO 标签、超宽黑边画幅和偏位乳腺图被放大进生成结果。
- 增加暗背景亮标记涂补函数，在生成前和最终保存前处理背景区域中的字母/金属小标记；乳腺组织区域会被保护，减少误伤解剖结构。

### Gradio 评估展示优化

- 修复评估汇总读取旧字段的问题：`summary.json` 当前写入 `count` 与 `group_means`，前端已同步读取，避免图像数与 A-F 分组分数显示为 0。
- 「评估」页改为答辩友好的结论展示：评估完成后直接显示通过率、均分、BRISQUE、功率谱 β、A-F 分组得分、Tier 分布和主要扣分项。
- 评估日志默认折叠，避免老师优先看到命令行输出；同时增加按评估结果排序的图片预览。
- 评估输出目录支持留空自动生成，避免多次评估覆盖同一个 `review_output_gradio` 目录。

### 第二轮图像质量调优：去除几何伪影与标注残影

- 定位几何形/菱形/漏斗状灰块的主要来源为生成后 `cv2.inpaint(..., TELEA)` 对组织内亮点和背景标记的修补痕迹；已移除组织内部 TELEA 修补。
- 将圆形 marker 和背景文字处理改为局部柔性压低/中值融合，不再使用会产生规则几何块的 inpaint 算法。
- 源图选择阶段新增标注与白点降权：背景文字、LMLO/RMLO 字样、金属点较多的源图会排到后面；几何比例和主体面积异常的源图仍直接剔除。
- 验证批次 `artifact_final_verify_20260514_174813` 已生成 4 张图：未再出现大面积菱形/漏斗状修补块，源图标注明显减少。
- 小白点处理改为更温和的孤立高亮点软化，仅作用于很小的连通域，避免把正常乳腺纹理抹成块状。
