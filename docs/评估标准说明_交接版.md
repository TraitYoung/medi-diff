## 0. 总览

**五个维度、16 个指标、每图一个 0-100 总分**。评估分三层：

1. **原始指标层**：从单张图抽 16 个数值（面积比、圆形度、轮廓曲率、熵、圆形密度等）。
2. **打分层**：每个指标映射成 0–1 的小分；严重偏离真实分布时打 tag。
3. **聚合层**：16 个小分 → 5 个维度组分 → 加权总分 → **一票否决 + 无硬伤 + 总分 ≥50 → ok=True**。

**关键设计**：所有阈值和评分中心值都是**从真实钼靶数据集（CBIS-DDSM, 10237 张 JPEG）自动校准**出来的，不是人工拍的。

```
真实图 ─┬─> 逐图提取 14 指标
        ├─> 计算每指标的 p1/p5/p50/p95/p99 作为基线分位数
        └─> 计算每张真实图的直方图/功率谱到基线均值的距离分布
                             │
                             ▼
                把 percentiles 写入 baseline.json
                             │
生成图 ─┬─> 加载 baseline.json，自动校准阈值（硬阈值 p1/p99、软带 p5/p95）
        ├─> 逐图提取同样 16 指标
        ├─> 每项打 0-1 分 + 可能的 tag
        ├─> 按维度加权聚合得总分
        └─> (总分≥50) ∧ (无 hard-tag) ∧ (各维度 ≥ --veto-group-min) => ok=True
```

## 1. 数据与预处理

- **输入**：单通道灰度 PNG/JPEG，`dtype=uint8`，范围 `[0, 255]`。
- **缩放**：读入后按长边 `--resize-long-side 1024` 等比缩放（FFT/卷积成本是 O(N²)，缩一下不影响统计稳定性）。
- **乳腺掩膜 `build_mask(gray)`**：
  1. 5×5 高斯模糊
  2. Otsu 二值化；前景超过 80% 则反相
  3. 椭圆核 7×7 MORPH_OPEN 1 次 + MORPH_CLOSE 2 次
  4. 取最大连通域
- **基线采样**：10237 张真实图随机抽 `--max-baseline-samples 500`（`seed=2026`），多进程并行 8 进程；实测 ~3 分钟建好基线。

## 2. 14 项原始指标及其计算公式

记号：`gray ∈ [0,255]^{H×W}` 为单通道灰度图；`mask` 为二值乳腺掩膜；`vals = gray[mask>0]`，再归一化到 `[0,1]` 得 `v_norm`。

### A. 构图与解剖（4 项）

| 指标 | 公式 / 算法 | 单位 |
|---|---|---|
| **A1 mask_ratio** | `count(mask>0) / (H·W)` | 比例 |
| **A2 circularity** | 对最大轮廓，`4π·A / P²`（A=面积，P=周长） | 无量纲，完美圆=1 |
| **A3 pectoral_line** | MLO 位胸肌线检测：<br>① 比较 mask 左右 1/3 密度，取密度更高侧为胸壁侧<br>② 该侧 ROI 做 Otsu + Canny（阈值 50/150）<br>③ HoughLinesP（`minLineLength=max(80, H/6)`，`maxLineGap=12`）<br>④ 角度落在 [15°, 75°] 的线段 ≥ 2 条记为 found | 布尔 + 线段数 |
| **A4-a contour_concavity** | 最外轮廓凸性缺陷最大深度：`cnt = largest contour`，`hull = convexHull(cnt, returnPoints=False)`，`defects = convexityDefects(cnt, hull)`，取 `max(defects[:, 0, 3]) / 256` 得最大像素深度；再除以 `sqrt(mask_area)` 归一化掉分辨率影响。 | 无量纲。真实下垂乳房凸面结构深度极小（p50 ≈ 0.05），被"咬一口"时跃升 0.3+ |
| **A4-b contour_perim_ratio** | `arcLength(cnt) / arcLength(convexHull(cnt))`。凸曲线 ≈ 1.0；锯齿/震荡时比值显著 > 1.2。 | 无量纲，≥ 1 |

### B. 灰阶分布（3 项）

| 指标 | 公式 | 单位 |
|---|---|---|
| **B1-a mean_intensity** | `mean(v_norm)` | `[0,1]` |
| **B1-b std_intensity** | `std(v_norm)` | `[0,1]` |
| **B2 dynamic_range** | `percentile(v_norm, 99) - percentile(v_norm, 1)` | `[0,1]` |
| **B3-a bright_ratio** | `count(v_norm > 0.90) / size(vals)` | 比例 |
| **B3-b dark_ratio** | `count(v_norm < 0.10) / size(vals)` | 比例 |

### C. 纹理与频率（2 项）

| 指标 | 公式 | 说明 |
|---|---|---|
| **C1 ps_emd** | 径向平均功率谱（64 bins）：`x = (gray - mean) / std`，`x = x·mask`，`F = fftshift(fft2(x))`，`P = |F|²`，按径向环 bin 均值 + `log1p` + 归一化。生成图 profile 与真实基线均值 profile 的 1D EMD（`mean(|cumsum(p) - cumsum(q)|)`）。 | 距离，越小越像真实 |
| **C2 local_entropy** | 5×5 局部方差图 `local_var = blur(g²) - blur(g)²`；取掩膜内所有 local_var 值；做 50-bin 直方图（`density=False`），归一化成概率质量 `p`；Shannon 熵 `-Σ p·ln p`。 | nats；真实 p50 ≈ 1.3 |

### D. 伪影与幻觉（4 项）

| 指标 | 公式 | 触发条件 |
|---|---|---|
| **D1 round_isolated** | 在乳腺区内，以 `vals` 的 85 分位做阈值提取亮区 → 3×3 开运算 → 连通域。保留 area∈[20,800]、长短轴比>0.6 的近圆形；任一圆心 60px 内无同类 → 记为孤立。 | 零散空泡扩散幻觉 |
| **D2 edge_voids_px** | mask 5×5 椭圆膨胀得 `dil`；`band = dil XOR mask`（掩膜外约 2px 的环带）；该环带内 `gray < 13` 的像素计数。 | 乳腺边缘纯黑带 |
| **D3 banding_score** | 只看掩膜内的行列条带：<br>① 在掩膜包围盒内，逐行/列只用 mask 内像素计算方差<br>② 记 `r_med`/`c_med` 为中位数<br>③ 异常行列数 = `count(var < 0.15·med) + count(var > 6·med)`<br>④ `score = max(0, 1 - total / max(20, 0.05·(行数+列数)))` | 越接近 1 越健康 |
| **D4 round_density** | 同 D1 提取圆形亮连通域，**不再判孤立**；汇总所有保留连通域的像素面积 / 乳腺面积。即"圆形结构"在腺体内的密度比例。 | 密集气泡阵列 / 马赛克网格（trypophobia） |

### E. 真实分布偏离（2 项）

| 指标 | 公式 | 说明 |
|---|---|---|
| **E1 hist_wass** | 乳腺内像素归一化到 `[0,1]`，100-bin 直方图（`density=True` 再重归一化为概率质量）。生成图 hist 与真实基线均值 hist 的 1D Wasserstein（scipy `wasserstein_distance`，支撑取 `linspace(0,1,100)`，权重为各自概率质量）。 | 距离，越小越像真实 |
| **E2 density_type** | 基于 `mean_norm` / `std_norm` 的 BI-RADS 风格四分型：<br>- `a_fatty`: mean<0.33 & std<0.10<br>- `b_scattered`: mean<0.45 & std<0.12（其他 fallback）<br>- `c_hetero`: std≥0.14 或 mean≥0.55<br>- `d_extreme`: std≥0.16 & mean≥0.55<br>与真实分布的 `density_prior`（真实四分型占比）做匹配度打分。 | 离散分类 |

### F. 学术无参考指标（2 项，2026-05-09 更新）

| 指标 | 公式 | 评分阈值 |
|---|---|---|
| **F1 ps_slope** | 对径向平均功率谱在中频段（bin 3-20）做对数线性拟合，得到幂律斜率 β。Burgess (1999) 证实真实钼靶 β ∈ [2.0, 3.5]。 | [2.0, 3.8] 满分；<1.2 或 >5.0 零分；触发 `PS_SLOPE_LOW/HIGH` |
| **F2 BRISQUE** | 通过 `piq.brisque()`（mask-based，仅乳腺区域），基于 NSS 自然场景统计的盲参考质量分。0=最好。 | **视觉优先收紧**：≤20 满分；>60 零分；>45 触发 `HIGH_BRISQUE`。真实 CBIS 钼靶 BRISQUE ≈ 50-70（mask-based），生成图应明显优于真实基线。 |

## 3. 自动校准：阈值来自真实分布

**关键设计决策**：把"阈值是否合理"这件事从人工猜变成数据驱动。

### 3.1 建基线时计算分位数

对所有真实图，对每个指标 `X` 收集：`{p1, p5, p50, p95, p99, mean, std}`。

此外还额外收集：
- 每张真实图的直方图到基线均值直方图的 Wasserstein 距离分布（`hist_wass`）
- 每张真实图的 RAPS 到基线均值 RAPS 的 EMD 距离分布（`ps_emd`）

→ 落盘为 `real_baseline_stats.json` 的 `percentiles` 字段。

### 3.2 加载基线后自动覆写阈值（`apply_calibration`）

- **硬阈值（触发 tag）**：全部取 `p1 / p99`（例：`min_circularity = p1, max_circularity = p99`）
  - 设计理由：真实分布中最离群的 ~2% 才算"真的坏"，避免人工拍的阈值误杀真实图
- **软带（给满分）**：取 `p5 / p95`（例：`_cal_circ_soft_lo = p5, _cal_circ_soft_hi = p95`）
- **评分中心值**：某些连续评分（B1 均值、B2 动态范围、C2 熵）以真实 `p50` 为中心，斜率用 `p95-p5` 的宽度做归一化
- **距离评分尺度**（C1 ps_emd、E1 hist_wass）：以真实 `p99` 作为"得 0 分"的距离，`score = clip(1 - dist / p99, 0, 1)`

### 3.3 自检验证

用这套阈值反过来评真实图自己：**通过率 91.5%**（理论上限 ~98%，剩下 ~8% 是真实数据集里的离群个例：截断图、残缺图、数据录入异常等）。这一步叫"尺子校正"，通过后才能用于评估生成图。

## 4. 每项 0–1 小分的评分逻辑

全部打分函数定义在 `grade_a1` ~ `grade_e2`。核心模式：

### 4.1 双档区间型（A1 / A2）

```
if soft_lo <= x <= soft_hi:          # 健康带（真实 p5~p95）
    return 1.0, ""
elif hard_lo <= x <= hard_hi:         # 临界带（真实 p1~p5 或 p95~p99）
    return 0.6, "*_BORDER"            # 非 hard-tag，不影响 ok 判定
else:                                 # 超出 p1/p99
    return ~0, "*_ODD / *_BAD"        # hard-tag
```

### 4.2 单边硬限型（B3 / D1 / D2）

```
if x > cal_p99:
    tag = "BLOWOUT / ARTIFACT_BUBBLES / EDGE_VOIDS"
score = clip(1 - max(0, x - 0.5·budget) / budget, 0, 1)  # budget = cal_p99
```

### 4.3 中心-散布型（B1 / B2 / C2）

```
score = mean(clip(1 - |x - cal_p50| / (cal_span / 2), 0, 1)  for each sub-metric)
if x < cal_p1: tag = "LOW_*"
if x > cal_p99: tag = "OVEREXPOSED / OVER_STRETCHED / TOO_NOISY"
```

### 4.4 距离衰减型（C1 / E1）

```
dist = emd(profile, ref_mean)             # 或 wasserstein 直方图
score = clip(1 - dist / cal_p99_dist, 0, 1)
if dist > cal_p99_dist: tag = "TEXTURE_UNNATURAL / HIST_OFF_REAL"
```

### 4.5 直接比例型（D3）

```
score = banding_score                      # 已 ∈ [0, 1]
if banding_score < cal_p1_banding: tag = "BANDING"
```

### 4.6 分类先验型（E2）

```
prior = real_density_prior[gen_density_type]
if prior ≥ 0.20: score = 1.0
elif prior ≥ 0.05: score = 0.6
else: score = 0.3, tag = "DENSITY_MISMATCH"
```

## 5. 聚合：维度组分 → 总分 → ok 判定

### 5.1 六个维度组分

```
group_A = mean(A1, A2, A3, A4, A5, A6, A7)   # 含竖向结构/解剖语义/CLIP
group_B = mean(B1, B2, B3)
group_C = mean(C1, C2, C3)                     # 含梯度纹理 C3
group_D = mean(D1, D2, D3, D4, D5)            # 含网格周期 D5
group_E = mean(E1, E2)
group_F = mean(F1, F2)                         # 功率谱斜率 + BRISQUE
```

**视觉优先约束（2026-05-09）**：当 `edge_density < 0.005` 时，C 组分数上限 0.5，防止过平滑图像通过其他纹理指标获得虚高分数。

### 5.2 加权总分（默认权重，2026-05-09 更新）

```
total_score = 100 * (0.20·A + 0.10·B + 0.20·C + 0.25·D + 0.05·E + 0.20·F)
```

权重设计原则：视觉质量组（C 纹理 + D 伪影 + F 感知）合计 65%，降低纯统计分布匹配（B 灰度 + E 分布偏差）至 15%。权重可通过 `--weights '{"A":0.3,...}'` 覆盖。

### 5.3 Pass / Fail 判定（v2：一票否决制）

```python
hard_tags = {
    "AREA_BAD",               # A1
    "SHAPE_ODD",              # A2
    "CONTOUR_FRACTURED",      # A4（轮廓断裂/锯齿）
    "BLOWOUT",                # B3
    "DEAD_DARK",              # B3
    "ARTIFACT_BUBBLES",       # D1
    "ARTIFACT_TRYPOPHOBIA",   # D4（密集圆形伪影）
    "EDGE_VOIDS",             # D2
    "BANDING",                # D3
    "NO_PECTORAL",            # A3 仅在 --require_pectoral 时
    "ANATOMY_NON_MAMMO",      # 跨器官/拼图式结构
    "CLIP_CROSS_ANATOMY",     # CLIP 跨解剖信号
    "MODALITY_LOW",           # 模态分类器低置信度
    "MASK_AREA_OUT_OF_DIST",  # 乳腺面积异常
    "SKIN_LINE_MISSING",      # 皮肤线缺失
    "CLOSED_RING_ARTIFACT",   # 闭合环形伪影
    "CENTER_OVEREXPOSED",     # 中心过曝
    "GRID_SEAM",              # Patch 拼接格线
}

veto = any(group_mean[g] < args.veto_group_min for g in "ABCDEF")   # 默认 0.30

ok = (total_score >= 50) and (no hard_tag triggered) and (not veto)
```

**一票否决设计**：当某维度组分 < 0.30（真实分布里极端离群段），即便总分靠别的维度拉到 60+，仍然判 Fail。避免"平均分掩盖局部致命缺陷"（例如：B/C/E 都正常，但 A=0.2 意味着乳房形状已崩坏，这张图必须拦下）。

**关闭方式**：`--veto-group-min 0` 退化回 v1 纯加权总分判定。

**注意**：`AREA_BORDER / SHAPE_BORDER` 是软 tag，会影响分数但不会令 ok=False。

## 7. 输出

每次 review 生成四个产物：

1. **`review_report.csv`** —— 逐图多列（ok / total_score / tags / veto / 5 维度分 / 15 项小分 / 原始指标 / legacy_reasons）
2. **`top_k_recommended.txt`** —— 按总分排序的推荐前 K 张
3. **`summary.json`** —— 批次级统计：通过率、平均总分、各维度均分、各 tag 违规率、**一票否决率（总体 + 分维度）**、阈值、权重、是否启用 auto-calibrate
4. **`real_baseline_stats.json`**（首次跑时）—— 10 个均值、10 个标准差、64 维 ps_mean、100 维 hist_mean、密度先验、所有 percentiles（v2 新增 `round_density`/`contour_concavity`/`contour_perim_ratio`）、`count`

`scripts/evaluation/compare_runs.py` 读取多个 `summary.json` 生成对比表（`compare_table.csv` / `.md` / 终端 leaderboard）。

## 8. 已知的建模局限（供 Gemini 质疑）

把最弱的地方主动列出来，方便外审审视：

1. **A3 胸肌线检测**：靠 Hough 直线，在生成图上可能被过度平滑或高光条反复触发；当前仅作 found/not-found 二值，未作 ROC 验证。
2. **C2 局部熵**：窗口硬编 5×5；不同缩放比例下 local variance 的绝对值尺度不同。目前靠 `--resize-long-side` 统一到 1024 避免。
3. **D1 孤立圆形区 / D4 圆形密度**：`area ∈ [20, 800]`、`aspect > 0.6`、孤立判定用 60 px 距离，这些参数未从真实基线自动校准（只有阈值/密度本身通过 p99 校准）。D4 对均匀腺体纹理里"真的有很多小钙化"的图（例：小结节病例）有 false positive 风险，但这类图原本也应归类为异常 → 拦下不算误伤。
4. **D3 条带检测**：阈值 `0.15·median`、`6·median`、`denom = max(20, 0.05·(行+列))` 均为经验值；如果真实图本身某些 ROI 方差分布奇特，会误判。
5. **E1 Wasserstein-1D**：支撑取 `linspace(0, 1, 100)`，100 bins 的 emd 与真实像素距离的对应关系是线性的，这个假设在极端灰阶下可能欠准。
6. **E2 density_type**：四分型的切分是拍的（类似 BI-RADS），没有做聚类验证；`density_prior` 只保证"真实里多的型算对"，不代表生成图每张都该归到多数型。
7. **Pass/Fail 阈值**：`total≥50 & no hard-tag` 是启发式，hard-tags 集合是人工挑的。若要严格意义下的"和真实图不可区分"，还应附带 two-sample 假设检验（目前没做）。
8. **单位一致性**：所有灰度计算都在 uint8 上，所有归一化都 `/ 255`；通道一律假设灰度（彩色图会在读入时被 `IMREAD_GRAYSCALE` 展平）。
9. **MLO vs CC 位不区分**：A3 对 CC 位会天然 found=False，当前默认只"温和扣分"（打 0.7 分）不打 tag，避免把所有 CC 位都判坏。
10. **采样偏差**：`--max-baseline-samples 500` 随机抽样，若数据集内某位（如 MLO_LEFT）占比不均，可能影响分位数；目前 seed 固定，可复现。

## 9. 当前实测结果（CBIS-DDSM + SDXL-img2img）

- **真实图自测**：10237 张 → 通过率 91.5%，平均总分 71.4，各维度 A=0.87/B=0.63/C=0.57/D=0.76/E=0.65
- **生成图（100 张，letterbox + bg 噪声 + strength 0.22-0.28）**：通过率 84%，平均总分 70.7，A=0.84/B=0.65/C=0.58/D=0.74/E=0.66
- **唯一系统性差距**在维度 A：`SHAPE_BORDER+SHAPE_ODD` 合计 26%（真实 10.8%）。归因：低 strength SDXL 的 VAE encode-decode 对乳腺边缘造成毫米级平滑化，使 circularity 略偏。

## 9.5 v2 升级点（相对 v1）

| 项 | v1 | v2 | 动机 |
|---|---|---|---|
| 指标数 | 14（A×3, D×3） | 16（A×4, D×4） | 加硬核结构指标 |
| A4 | — | `contour_concavity` + `contour_perim_ratio` → `CONTOUR_FRACTURED` | 抓"被咬一口"式轮廓崩坏（图二类） |
| D4 | — | `round_density` → `ARTIFACT_TRYPOPHOBIA` | 抓"气泡阵列 / 密集网格"（图三类） |
| D1 | 孤立圆形数 → `ARTIFACT_BUBBLES` | **不变**，保留零散空泡检测 | 同时开 D1 + D4 形成双通道防御 |
| Pass/Fail | 总分 ≥50 且无 hard-tag | 再加"任一维度 ≥ --veto-group-min（默认 0.30）" | 防止平均分掩盖局部崩坏 |
| 校准 | 12 个 percentile | 15 个 percentile | 新增 3 个硬阈值用真实 p99 |
| summary.json | tags 违规率 | + `veto_total` / `veto_rate` / `veto_by_group` | 诊断哪个维度是当前最大短板 |

**v2 需要重建基线**：旧的 `real_baseline_stats.json` 没有 `round_density / contour_concavity / contour_perim_ratio` 的 percentiles，加载后这 3 项会 fallback 到手写默认值（`0.04 / 0.35 / 1.35`），虽然能跑但阈值不是数据驱动。建议删除旧 baseline，`--real-images-dir` 重新算一次（10237 张采样 500 张，多进程 8 线程，~3 分钟）。

## 10. 关键代码定位

- `extract_metrics(gray)` —— 一次性算 16 个原始指标，返回 dict
- `detect_isolated_round(gray, mask)` —— 返回 `(suspicious, isolated, total_area, density)`，同时供 D1 / D4 用
- `compute_contour_irregularity(mask)` —— 返回 `(concavity_norm, perim_ratio)`，供 A4 用
- `build_reference_stats(real_dir)` —— 建基线（均值 + percentiles + 距离分布）
- `apply_calibration(args, ref)` —— 从 percentiles 覆写阈值与评分中心值（v2 新增 3 个硬阈值 + 3 个软尺度）
- `grade_a1 ~ grade_e2`（含 `grade_a4` / `grade_d4`）—— 15 个打分函数
- `score_image(m, ref, args)` —— 聚合为 5 维度 + 总分 + **一票否决 veto** + ok + tags
- `main()` —— 批量处理 + 写 CSV/top-k/summary/baseline

脚本 CLI 最小调用：

```bash
python scripts/evaluation/review_generated_images.py \
    --images-dir <gen_dir> \
    --real-baseline-json <real_baseline_stats.json> \
    --output-dir <out_dir> \
    --resize-long-side 1024 \
    --review-workers 0
```
