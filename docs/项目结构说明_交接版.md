## 根目录总览

```text
/root/autodl-tmp/medi-diff
├─ apps/                 # FastAPI、Gradio；说明见 apps/README.md
├─ scripts/              # 预处理、生成、训练、评估、批量实验

├─ README.md
```

## 常用命令（主线 SD1.5）

生成（示例，请按本机填写 `--base-model-local` 或改用在线模型 ID）：

```bash
python3 scripts/generation/run_mammo_sd15.py \
  --num-images 20 \
  --strength 0.42 \
  --lora-path outputs/lora/mammo_sd15_v4_clean/final_lora \
  --metadata-csv datasets/CBIS_CLEAN_V2/metadata_clean.csv \
  --scheduler dpm --num-steps 20 --patch-size 640 \
  --output-subdir-prefix sd15_batch
```

**按 CBIS 标签筛选源图**（与全目录随机二选一；**推荐 V2 CSV 与 v4 LoRA 对齐**）：

```bash
python3 scripts/generation/run_mammo_sd15.py \
  --metadata-csv datasets/CBIS_CLEAN_V2/metadata_clean.csv \
  --filter-view MLO --filter-density dense \
  --num-images 8 \
  --lora-path outputs/lora/mammo_sd15_v4_clean/final_lora \
  --base-model-local hf_cache/sd15 \
  --scheduler dpm --num-steps 20 --patch-size 640 \
  --output-subdir-prefix sd15_tagged
```

**全自动：生成 + 评审 + 可选 NLP/VL 报告 + 汇总 Markdown**：

```bash
bash scripts/assistant/start_full_report.sh
# 或带标签 + 调参 JSON（示例）：
python3 scripts/assistant/run_full_report.py \
  --metadata-csv datasets/CBIS_CLEAN_V2/metadata_clean.csv \
  --filter-view MLO --filter-density dense --num-images 6 \
  --tuning-json outputs/reports/tuning_round7_seam_focus.json \
  --advisor-mode both
```

- **`--advisor-mode`**：`both`（先文本 REST API，必要时再调 **Qwen‑VL**）、`text_only`、`vision_only`；未配置密钥时评估与生成仍可进行。  
- **`outputs/reports/<tag>/`**：除 **`advisor_suggestions.md`** 外，`both` 模式下可产生 **`advisor_qwen_text.md`**、**`advisor_qwen_vision.md`**。  
- **生成侧透传**：`--overlap-ratio`、`--strength`、`--global-guide-strength`、`--global-guide-blend`、`--num-steps`、**`--blend-sigma-divisor`**、**`--guidance-scale`** 等会传给 `run_mammo_sd15.py`（详见脚本 `--help`）。

评估：

```bash
python3 scripts/evaluation/review_generated_images.py \
  --images-dir "outputs/generated/毕业论文_生成图像/<sd15_某批次>" \
  --no-recursive \
  --output-dir outputs/eval/<名称>
```

多轮对比（扫描 `outputs/eval` 下各 `review_output_*` 或整目录内 summary）：

```bash
python3 scripts/evaluation/compare_runs.py --runs-dir outputs/eval
```

历史 SDXL 的 `review_output_*` 位于 `outputs/_legacy/sdxl/reviews/`，对比时需把 `--runs-dir` 指到该路径。
